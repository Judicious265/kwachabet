# KWACHA BET

**Malawi's Premier Online Betting Platform**

A full-featured sports betting web application built for Malawi. Place bets on football, basketball, tennis, cricket, ice hockey, rugby, MMA and more.

## Features

- Real-time odds that update live as games progress
- Live betting section with live scores
- Malawian phone number registration (+265 88x / +265 99x only)
- 4-digit PIN security for login and withdrawals
- Mobile wallet payments: Airtel Money, Mpamba (TNM), Paychangu
- Accumulator (multi-bet) builder with potential winnings calculator
- My Bets section: Open bets and Settled bets
- Wallet with full transaction history
- Promotions and bonuses
- Admin panel with customer manager and transaction records

## Hosting on GitHub Pages

1. Upload all files in this folder to your GitHub repository
2. Go to **Settings** > **Pages**
3. Under **Source**, select **Deploy from a branch**
4. Select **main** branch and **/ (root)** folder
5. Click **Save**
6. Your site will be live at: `https://yourusername.github.io/your-repo-name/`

## Files

| File | Purpose |
|------|---------|
| `index.html` | Main application (entire app is in this one file) |
| `404.html` | Handles 404 redirects for GitHub Pages |
| `_config.yml` | GitHub Pages configuration |
| `README.md` | This file |

## Important Note for GitHub Pages

If your repository is **not** at the root of a custom domain (e.g. it is at `yourusername.github.io/kwachabet/`), the app will still work fine because it is a single-page application with no routing.

## Technology

- Pure HTML, CSS and JavaScript (no build tools required)
- Google Fonts (Barlow Condensed, Inter, JetBrains Mono)
- No external dependencies or npm packages needed
- Works offline after first load
